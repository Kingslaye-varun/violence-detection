================================================================================
                    TROUBLESHOOTING
================================================================================

Issue: "Could not open webcam"
Fix: Try different camera index
     cap = cv2.VideoCapture(1)  # Change 0 to 1, 2, etc.

Issue: Gender not showing
Fix: Check if gender_classification.tflite exists
     If missing, gender detection will be skipped

Issue: No skeleton visible
Fix: pip install --upgrade mediapipe

Issue: Streamlit not opening
Fix: streamlit run streamlit_dashboard.py --server.port 8501

Issue: Low FPS
Fix: Increase FRAME_SKIP in code
     Or use lower webcam resolution

Issue: Too many false alarms
Fix: Increase VIOLENCE_THRESHOLD from 0.7 to 0.8
